# Pràctica I

## Descripció

Aquesta pràctica correspon als mòduls **0485 Programació** (RA1, RA3, RA7) i **0487 Entorns de Desenvolupament** (RA4). L'objectiu és implementar un software que resolgui una idea o necessitat del món real, utilitzant les habilitats de programació lineal vistes a classe (sense fer ús de tècniques no vistes a classe).

El projecte es desenvolupa en grup de forma autònoma en Java usant IntelliJ.

## Requisits Tècnics

### Programació (0485)
- Algoritmes dissenyats i implementats de manera eficient
- Menú d'usuari amb opció de sortir
- Ús de consola per interactuar amb l'usuari amb control d'errors
- Estructures condicionals (IF-ELSE, SWITCH)
- Estructures iteratives (WHILE, DO-WHILE, FOR)
- Col·leccions estàtiques (vectors i matrius)
- Col·leccions dinàmiques (ArrayLists)
- Mètodes per encapsular funcionalitats

### Entorns de Desenvolupament (0487)
- Ús del repositori de GitHub de classe
- Control de versions correcte
- Creació de branques per funcionalitat
- Eines de gestió de projectes
- Creació d'una Release per a l'entrega final

## Entregables

1. Diagrames fets a la validació de la idea
2. Release del projecte
3. Presentació

## Recordatoris Importants

1. **Modularitat**: El `main` només ha de contenir el bucle del menú principal (`do-while` + `switch`). Tota la lògica ha d'estar encapsulada en funcions estàtiques.

2. **Validació d'Entrada**: Crear funcions auxiliars per evitar errors quan l'usuari introdueix dades incorrectes.

3. **Estil de codi**: Noms de variables descriptius i comentaris JavaDoc a les funcions.
