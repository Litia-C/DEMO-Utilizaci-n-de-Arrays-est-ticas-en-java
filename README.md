# Selector de series

## Descripción:
Este proyecto es una prueba hecha como trabajo de clase para utilizar conocimientos básicos de java, en este caso se utiliza una array para guardar diferente información para tener un registro de series y poder ordenarlas. 
Se realizó en conjunto con una compañera de clase.
